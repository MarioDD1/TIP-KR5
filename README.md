# Мини-игра "Угадай число" на Express.js

Контрольная работа по Express.js, демонстрирующая принципы работы с фреймворком, создание сервера, маршрутов, middleware и структурирование backend-приложения.

## Функционал приложения

1. **RESTful API** для игры "Угадай число"
2. **Middleware** для логирования и валидации
3. **Статические файлы** (HTML, JS) для фронтенда
4. **Структурированная архитектура** (routes, controllers, middleware)
